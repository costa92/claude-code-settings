# Blog Digest — Markdown 输出格式

与 news-daily 保持一致的标准格式。

---

## 模板

```markdown
# Blog Digest · {年}年{月}月{日}日

> 来自 Karpathy 推荐的 N 个顶级技术博客 · 扫描 N 源 · 精选 N 篇

## 核心摘要

- {3-5 条要点，每条一句话，以"- "开头}

---

## 📊 数据概览

| 扫描源 | 抓取文章 | 时间范围 | 精选 |
|:---:|:---:|:---:|:---:|
| N/N | N 篇 → N 篇 | Nh | **N 篇** |

### 分类分布
(Mermaid pie chart)

### 高频关键词
(Mermaid xychart-beta)

### 🏷️ 话题标签

---

## {emoji} {分类名称}

### {中文标题}

{详细摘要，翻译成中文}

**关键信息**: {标签1, 标签2, 标签3}

📎 原文链接: [{来源域名}]({原始URL})

---

数据来源: Karpathy 推荐 N 个技术博客 RSS（source1, source2 等）
生成时间: YYYY-MM-DD HH:MM
```

---

## 格式规范

| 元素 | 格式 |
|------|------|
| 标题 | `# Blog Digest · {年}年{月}月{日}日` |
| 核心摘要 | `## 核心摘要` + 分条要点（`- `开头） |
| 分类标题 | `## {emoji} {中文分类名}` |
| 文章标题 | `### {中文标题}`（无序号） |
| 正文摘要 | 纯段落，中文，不加 `>` 引用块 |
| 关键词 | `**关键信息**: tag1, tag2` |
| 原文链接 | `📎 原文链接: [来源域名](url)` |
| 页脚 | `数据来源: ...` + `生成时间: YYYY-MM-DD HH:MM` |

## 分类名称

| 分类 ID | 中文名 | Emoji |
|---------|--------|-------|
| ai_ml | AI / ML | 🤖 |
| security | 安全 | 🔒 |
| engineering | 工程 | ⚙️ |
| tools | 工具 / 开源 | 🛠 |
| opinion | 观点 / 杂谈 | 💡 |
| other | 其他 | 📝 |
