# Plan: Add Kratos Framework Support (HTTP + gRPC Native Mode)

## Context

The `osbuilder-templates` project currently supports `gin`, `grpc`, and `grpc-gateway` as web frameworks. The user needs to add `kratos` as a new `webFramework` option. Kratos will use its native mode: **both HTTP and gRPC transports** running simultaneously, with APIs defined via proto + `google.api.http` annotations, and handler code generated by `protoc-gen-go-http`.

Since Kratos handlers implement the same proto service interfaces as gRPC handlers (method signature: `Method(ctx, *Request) (*Response, error)`), the handler code will follow the `handler/grpc/` pattern closely. The major differences are in the server setup (using Kratos transport layer) and middleware (using Kratos middleware chain).

## Middleware Design (Kratos Pattern)

Kratos middleware signature:
```go
func MyMiddleware() middleware.Middleware {
    return func(handler middleware.Handler) middleware.Handler {
        return func(ctx context.Context, req interface{}) (interface{}, error) {
            // before
            reply, err := handler(ctx, req)
            // after
            return reply, err
        }
    }
}
```

For auth middleware, use Kratos's `transport.FromServerContext(ctx)` to extract HTTP headers (for JWT token).

Auth whitelist for Kratos uses `selector.Server()` middleware from `github.com/go-kratos/kratos/v2/middleware/selector`.

## kratosserver.go Design

```go
type kratosServer struct {
    app *kratos.App
}

func (c *ServerConfig) NewKratosServer() (server.Server, error) {
    // Build middleware chain
    middlewares := []middleware.Middleware{
        recovery.Recovery(),
        {{- if .Web.WithOTel}}
        tracing.Server(),
        {{- end}}
    }

    // HTTP server
    httpSrv := kratoshttp.NewServer(
        kratoshttp.Address(c.HTTPOptions.Addr),
        kratoshttp.Middleware(middlewares...),
    )
    v1.Register{{.Web.GRPCServiceName}}HTTPServer(httpSrv, c.Handler)

    // gRPC server
    grpcSrv := kratosgrpc.NewServer(
        kratosgrpc.Address(c.GRPCOptions.Addr),
        kratosgrpc.Middleware(middlewares...),
    )
    v1.Register{{.Web.GRPCServiceName}}Server(grpcSrv, c.Handler)

    // Kratos app
    app := kratos.New(
        kratos.Name(serviceName),
        kratos.Server(httpSrv, grpcSrv),
    )

    return &kratosServer{app: app}, nil
}

func (s *kratosServer) RunOrDie(ctx context.Context) {
    if err := s.app.Run(); err != nil {
        panic(err)
    }
}

func (s *kratosServer) GracefulStop(ctx context.Context) {
    s.app.Stop()
}
```
